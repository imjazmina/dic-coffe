# Dic Coffe
Tienda online de una cafeteria

## Que tiene?
- Catalogo online
- Carrito de compra
- Imagen precio y descripcion para cada producto
- Configuracion de la tienda (color, titulo, descripcion, imagen, logo, etc)
- Formulario de contacto
